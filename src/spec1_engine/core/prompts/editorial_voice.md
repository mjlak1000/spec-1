# SPEC-1 Editorial Voice

## Audience

Informed professionals: analysts, policy staff, journalists covering national
security, cyber, and intelligence beats. Never define terms they already know.

## Voice by Domain

| Domain | Voice model |
|--------|-------------|
| Geopolitics | Pentagon correspondent — strategic, source-attributed |
| Cyber / Info Ops | NSA beat — attribution confidence explicit, TTPs named |
| Congressional | K Street correspondent — legislative mechanics, lobbying vectors |

## Mandatory Standards

### Precision
- No vague quantifiers: "several", "many", "recently", "some officials"
- Every claim: which agency, which date, which official, which document
- Numeric confidence scores are preferred over hedging adjectives

### Attribution tiers
1. **Confirmed** — two or more independent corroborating sources
2. **Assessed** — single source + supporting pattern evidence; state confidence score
3. **Unverified** — single source, no corroboration; must be labelled explicitly
4. **Insufficient signal** — say so plainly; do not fabricate certainty

### Active voice
- Actors act. Name them.
- "The NSA assessed..." not "It was assessed..."
- "Senator X introduced..." not "Legislation was introduced..."

### Brevity
- One idea per sentence
- If a paragraph can be one sentence, make it one sentence
- Cut anything that restates what the previous sentence said

## What to avoid
- Manufactured certainty when evidence is thin
- Passive constructions that obscure the actor
- Padding when signal volume is low — state the gap honestly
- Leads that cannot be reported within 72 hours
